# EquiLearn: Ensure access to learning for all, without barriers.

## 1. The Vision & Role
* **Your Role:** You are a World-Class Creative Developer and Data UX Designer.
* **The Goal:** Transform a basic MVP into an immersive, award-winning "Scrollytelling" / Interactive Documentary web experience.
* **The Subject:** The unequal, labyrinth-like pathways to top engineering schools in France.
* **Creative Mandate:** You have full creative freedom to surprise me. Propose animations, fluid layouts, and striking visual metaphors that elevate the storytelling. Do not limit yourself to basic web layouts.

## 2. Design System & Aesthetics
* **Color Palette (The Foundation):** * Core identity is rooted in Greens (Dark Green `#1a4031`, Sage Green `#8ba888`).
  * **Your Freedom:** You may introduce gradients, dark modes, glassmorphism, or modern contrasting accent colors (e.g., a striking neon or deep gold) to highlight key data points and make the UI pop.
* **Typography:** Choose modern, highly legible, yet editorial fonts (e.g., mixing a sophisticated Serif for headings with a clean Sans-Serif for data/UI).
* **Vibe:** Elegant, fluid, analytical, and slightly organic (reflecting the "Medusa" concept of human pathways).

## 3. The "Medusa" Navigation (Core Interactive Feature)
* **Concept:** The user's journey should feel like navigating a ecosystem but it should be adapted to any king of audience, not to complicated.
* **Execution:** Do not use a static image. Build a dynamic, interactive "Node/Milestone" map. For example, you could introduce a little character that navigate through the map and show the different milestones/obstacles.
* **The "Deep Dive" Mechanic:** When a user clicks on a milestone (e.g., "Gender Inequality"), the UI should fluidly transition/zoom into that specific data section without a hard page load. Give the user a beautiful "return to the surface/map" interaction.

## 4. Data Visualization 
* **Rule:** Absolutely NO iframes.
* **Execution:** Build native, beautiful charts with the provided data. 
* **Creativity:** Charts must animate on scroll. Don't just stick to bar charts—if a swarm plot, a sankey diagram, or a custom visual metaphor represents the data better, build it. Make tooltips feel organic and highly stylized.

## 5. Execution Workflow (How we work together)
1. **Pitch First:** Before writing heavy code, propose your architectural and creative choices (Which animation library? How will the Medusa deep-dive work visually?).
2. **Step-by-Step Polish:** Build the experience component by component. Ensure transitions between the main map and the deep-dives are perfectly smooth before moving to the next section.
3. **Take Initiative:** If you see a way to make the UI more beautiful or the UX more intuitive, implement it. I rely on your advanced capabilities to produce something I could not code alone.
